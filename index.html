Uso y Aplicación de paginas web

1¿Que son las paginas web?

Se conoce como página web al documento que forma parte de un sitio web y que suele contar con enlaces (también conocidos como hipervínculos o links) para facilitar la navegación entre los contenido. las páginas pueden presentar información en distintos formatos (texto, imágenes, sonidos, videos, animaciones).

Tiene que tener un diseño atractivo para poder llamar la atención del usuario y conseguir que navegue por ella.

2Caracteristicas y Como se desarrollan

Las páginas web están desarrolladas con lenguajes de marcado como el HTML, que pueden ser interpretados por los navegadores. Entre las múltiples características que tiene una página web y que sirven para identificarla se encuentran las siguientes: cuenta con información textual y también con material de tipo audiovisual.

Un conjunto de páginas web, por lo tanto forman un sitio web y las páginas suelen estar reunidas bajo un dominio común para que el sitio en cuestión sea accesible desde una misma dirección en Internet.

¿Como son?

Estáticas

Solo desplegaran información, no permite hacer muchos movimientos, es muy insípido, es como leer un periódico y el periódico real tiene el efecto de darle la vuelta, algo que en una página web estática no se podria. Su única ventaja es su gran bajo costo.

Dinámicas

Invita al usuario a interactuar, pero para lograr eso se tiene que programar la página dinámica, con la tecnología que elijamos para darle funcionalidad y efectos. La creación de una página web dinámica es más compleja por que se requieren conocimientos muy adentrados en códigos para una buena programación. El 90% de los sitios webs trabajan sobre el esquema dinámico, donde el usuario tendrá interfaces llamativas y de fácil uso.

¿Donde y Como se aplican?

Escuela

Son fundamentales para que podamos realizar nuestras tareas, sin el apoyo deuna página web, nos enfocaríamos solo en libros o revistas y nuestras opciones de búsqueda e investigación serian un poco limitadas.

Negocio pequeño

Son importantes porque mediante una página web, puedes promocionar tus productos a más público y ya no depender de solo de folletos, periódicos o anuncios en la calle.

Empresa

Cuando ya es un empresa, las páginas web son de suma importancia porque no solo pueden ver los productos que ofreces sino que puedes enviarlos y a hacer que tu empresa crezca más.
